Spin & Win India 🇮🇳

Spin & Win India is a simple web-based game where users can spin a wheel and win coins.
The project is built using basic web technologies like HTML, CSS, and JavaScript.

Features:

- Spin Wheel Game
- Coin System
- Daily Bonus
- Simple Popup Reward System
- Mobile Friendly Design

Project Files:

- index.html – Main webpage of the game
- style.css – Styling and layout design
- script.js – Game logic and spin functionality
- swilogo.png – Logo used in the header
- medha.png – Background image used for design

How to Play:

1. Open the website.
2. Click the SPIN button.
3. Each spin costs 10 coins.
4. The wheel will rotate and randomly give a reward.
5. Coins will automatically update.
